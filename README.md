# Repositório de Aperfeiçoamento Profissional em Dados
Este repositório reúne meus estudos, experimentos e projetos aplicados na área de Ciência de Dados, Machine Learning e Analytics, com foco em desenvolvimento técnico contínuo e aplicação prática orientada a negócios.

O objetivo é consolidar conhecimento por meio de:

📌 Construção e avaliação de modelos preditivos

📌 Tratamento, exploração e engenharia de dados

📌 Técnicas de balanceamento e otimização de métricas (ex: foco em recall para classes minoritárias)

📌 Implementação de algoritmos como Regressão Logística, Random Forest e XGBoost

📌 Análise estatística aplicada e fundamentos probabilísticos

📌 Visualização e comunicação de resultados

Os projetos aqui desenvolvidos priorizam:

🔎 Rigor analítico

📈 Tomada de decisão orientada por dados

⚙️ Eficiência computacional

🎯 Aplicabilidade prática em cenários reais

Este repositório representa minha jornada de evolução técnica e aprofundamento em Data Science, sempre buscando unir fundamento teórico, experimentação prática e impacto real no negócio.
